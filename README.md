# 📖 Simulacra Codex (Finral's Version)

> "Uma ficha de personagem não é apenas papel ou código; é o espelho de uma alma. Esta é a minha versão definitiva, livre das amarras dos sistemas fechados." 
> — *Finral*

Uma ficha de personagem em HTML viva, personalizável e de código aberto. Criada para dar aos jogadores o controle total sobre suas identidades no RPG.

## 🚀 Como Usar
1. Faça o download ou clone este repositório.
2. Abra o arquivo index.html no seu navegador.
3. Preencha com os dados do seu personagem!
